# BioTrack solutions 

## Sobre o Projeto
O **BioTrack solutions** é um protótipo de sistema de informação desenvolvido no âmbito da unidade curricular de SIBDAS (Sistemas de Informação e Bases de Dados em Saúde) da Licenciatura em Engenharia Biomédica no ISEP. 

Este sistema foi concebido para mitigar os problemas de dispersão de dados na gestão do ciclo de vida de equipamentos hospitalares, servindo de alicerce visual para a futura implementação de um sistema GMAC (Gestão de Manutenção Assistida por Computador).

## Estrutura da Aplicação
O protótipo divide-se em duas áreas funcionais:
* **Área Pública (`/public`):** Apresentação institucional da empresa, detalhe de serviços e formulário de acesso.
* **Área Privada (`/private`):** Acesso reservado (após autenticação) ao dashboard estatístico interativo, listagem e registo de equipamentos, gestão de fornecedores e mapeamento hospitalar.
